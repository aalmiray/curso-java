# HolaMundo en Java (3)

## Conceptos

 - estructuras de control (if/else)
 - arreglos
 - excepciones

## Comandos

```
$ javac -d classes com/acme/*.java com/acme/greetings/*.java
$ cd classes
$ java com.acme.Main
```

Tambien puedes indicar en que idioma se debe ejecutar el programa

```
$ java com.acme.Main es
$ java com.acme.Main en
```
